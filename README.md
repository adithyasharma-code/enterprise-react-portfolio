## 📖 Overview

This portfolio was built to demonstrate **real-world frontend engineering** rather than simply present a résumé.

It combines modern React development, Tailwind CSS, reusable component architecture, and enterprise software engineering principles to create a recruiter-friendly and production-ready application.

The project also documents its complete engineering journey through a **30-chapter Engineering Handbook**.

# 💻 Local Development

Clone the repository

```bash
git clone <repository-url>
```

Navigate into the project

```bash
cd enterprise-react-portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
